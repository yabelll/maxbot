# Приключения Тюленюси и профессора Экосты

Приключения Тюленюси и профессора Экосты - чат-бот, написанный на TypeScript с использованием библиотеки @maxhub/max-bot-api для мессенджера Max.
Проект поддерживает запуск через Docker, автоматическую сборку и проверку кода через ESLint и Prettier.

## Структура проекта

В репозитории присутствуется каталог с написанным кодом для чат-бота на TypeScript, который содержит в себе логику сцен, подсчёт рейтинга и обработчики действий пользователя.

## Технологии
1. TypeScript — основной язык проекта
2. @maxhub/max-bot-api — библиотека для взаимодействия с мессенджером Max
3. Docker — для изоляции окружения и простого деплоя
4. ESLint и Prettier — для поддержания чистоты и единого стиля кода

## Требования к запуску

Перед запуском убедитесь, что у Вас установлены: Docker, Git

## Установка и запуск кода

1. Клонируйте репозиторий 
```bash
git clone https://github.com/yabelll/maxbot.git
cd maxbot
```

2. Запустите проект при помоще Docker с указанием вашего токена бота (BOT_TOKEN)
```bash
echo "BOT_TOKEN=your_bot_token" > .env; docker-compose up -d --build
```
